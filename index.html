Welcome to Oborozuki’s documentation!#

Oborozuki is a Python project utilising twitchio and twitch’s API to run channel moderation, with custom command capability and some fun mini-games. This project has just been released in closed beta, if you wish to participate, please head to Dot’s channel and type !invite into the channels chat!

Note

This project is under active development and is in EARLY stages of a beta

if you wish to give feedback or suggestions please head to here

Standard commands:#

Note

The prefixes for this bot are

! and ~

this allows any clashes to be avoided

Here is some usage on the standard commands to config this bot in your channel

!command

  • add/remove/edit/option - you can use this to modify the custom commands in your channel

  • add or a : use this to add a command, required args: !command add {name} {command response} I.E. !command add !hi hello o/ !

  • remove or r : use this to remove a command, required args: !command remove {name} I.E. !command remove !hi

  • edit or euse this to edit a command, required args: !command edit {name} {command response} I.E. !command edit !hi hello i am new here o/ !

    • if you just do !command edit {name} you will get back the raw command response to enable easy editing

  • option or o : use this to adjust the options for the command, there are 3 options to adjust; user level and global/user cooldowns (level=E,S,V,M,B)(GCD=0)(UCD=0) required args: !command option {name} {option} I.E. !command option !hi GCD=5

Note

  • this can also be called by !cmd

!quiet

  • use this to disable all commands in this channel

  • if you follow this command with a 1 this will disable the commands I.E. !quiet 1

  • if you follow this command with a 0 this will enable the commands I.E. !quiet 0

!links

  • use this to enable link protection in the channel

  • if you follow this command with a S=subscribers, V=vips, M=moderators this will enable the link protection for these people or higher to send links I.E. !links V will only allow VIPs or mods to send links

  • if you follow this command with a 0 this will disable the link protection I.E. !links 0

!permit

  • use this to allow someone to send a link as a one time exception

  • you need to follow this with a username.. I.E. !permit @Azuremoon13

!banned

  • use this to ban a certain word from your channel.

  • follow this command with the word you wish to ban I.E. !banned poo

  • can only ban one word at a time

!emotes

  • use this to enable emotes only mode - (useful for mobile moders)

!emotesoff

  • use this to disable emotes only mode - (useful for mobile moders)

!clear

  • This will purge the chat of all messages. (blocked by BTTV) - (useful for mobile moders)

!mso

  • This will give a default SO to ‘x’ amount of people. simple follow the command with as many @’s as you want seperated by a space

!modcheck

  • This will ping all active mods in chat asking if they are around!

Note

  • This all picks up mods once they have chatted in chat once!

!quote

  • add or a : use this to add a quote

  • remove or r : use this to remove a quote

  • if a number is supplied after this command, it will fetch that quote I.E !quote 6 = ‘Hello person! i am dot!’

  • if NO number is provided this will randomly select a quote for this channel

Mini-games:#

!hangman
  • This will allow you to play hangman as a group in chat!

    • once someone has started this game, others simple type ‘join’ in chat to participate

    • each round dot will ask for a letter suggestion, the letter that has been suggested most will be used as your guess

!rps

  • This is your classic game of Rock, Paper, Scissors, Lizard, Spock.. you will be playing dot! GL

!defuse

  • This is a bomb defusing mini-game, it will give you a series of wires to choose from, get it right live to tell your tale.. otherwise not so much!

!wins

  • you can use this command to find out how many wins or loses you have for any game, simple follow the command with the game I.E !wins hangman

Custom command args:#

${user}

This is to get the name of the user that invoked the command: ‘I am ${user}, hello’ = I am obrozuki13, hello

${game}

This is to return the current game of the channel: ‘We currently are playing ${game}’ = We currently are playing Valorant.

${target_url}

This is to return the target in a given command: ‘this is there link: ${target_url}’ = this is there link: www.twitch.tv/Azuremoon13

${target_game}

This is to return the target in a given command: ‘${user} is playing ${target_game}’ = Azure is playing Valorant

${target}

This is to return the target in a given command: ‘${user} hugs ${target}’ = obrozuki13 hugs azure

Note

  • if there is no user specified for the target, it will self target the user.

  • if there is no tag from twitch using ‘@’ it will default to the first word after the command invoke, conversely if there is a tag it will use this even if its not first word.

${mention}

This is to return the target in a given command but with the @: ‘hello ${mention}!’ = hello @dot_thebot!

${url.x}

This is to run a api fetch from a url and return it in plain text. ‘we currently are ${url.https://api.kyroskoh.xyz/valorant/v1/mmr/eu/x/x}’ = we currently are Radiant - 69rr

${count x}

This is used to create a count for a certain thing, this will increment the counter each time it is invoked: we have died ${count deaths} times = we have died 13 times

Note

  • this doesnt just have to me a random name, you can use other custom args in this to create unique counters

${getcount x}

This is used to get a count for a certain thing: we have died ${getcount deaths} times = we have died 13 times

${randuser}

This will return a random user in the chat. Here is a random user.. ${randuser} you have won! = Here is a random user.. Azure you have won!

Note

  • this will also include any bots in your chat.

  • they also have to of spoken in chat

${randmod}

This will return a random active moderator in the chat. Here is a random mod.. show your love to ${randmod}! = Here is a random mod.. show your love to Azure!

Note

  • this will also include any bots modded in your chat.

  • they also have to of spoken in chat

${randnum.x-y}

this will return a random number between the 2 numbers you specify. ‘I am ${randnum.0-110}% lit today’ = I am 69% lit today.

${randchoice.[x,y,z]}

this will return a random choice between the list you specify. ‘I am ${randchoice.[funny,cool,sexy]}’ = I am cool.

Note

-any items need to be between [ ] and seperated by a ,
